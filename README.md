# 🎓 EduFund - Student Micro-Scholarship DAO

A decentralized platform enabling community-driven micro-scholarships for students in need.

## 🌟 Features

- 💰 Anyone can donate STX to the fund
- 📝 Create scholarship proposals for students
- 🗳️ Community voting system
- ✅ Automated fund distribution
- 🔒 Secure and transparent

## 🚀 Smart Contract Functions

### For Donors
- `donate()`: Contribute STX to the scholarship fund

### For Students
- `create-proposal`: Submit a scholarship request
- `claim-funds`: Claim approved scholarship funds

### For Community
- `vote`: Vote on scholarship proposals
- `get-proposal`: View proposal details
- `get-total-funds`: Check total available funds
- `has-voted`: Check if an address has voted

## 📋 Usage Requirements

- Minimum proposal amount: 100,000 microSTX
- Proposal duration: 1440 blocks (~10 days)
- Required votes for approval: 3
- Proposals expire if not approved within duration

## 🔧 Technical Details

The contract is built on Stacks blockchain using Clarity language and supports:
- Secure fund management
- Automated proposal lifecycle
- Democratic voting system
- Transparent fund distribution
